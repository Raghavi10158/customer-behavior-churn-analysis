Customer Behavior & Churn Analysis (SQL Project)

## Overview
This project analyzes customer data to understand purchasing behavior, identify high-value customers, and uncover factors influencing churn.

## Objectives
* Identify high-value customers
* Analyze churn patterns
* Segment customers based on behavior
* Evaluate impact of discounts on purchases
* Compare loyal vs non-loyal customers

## Dataset
The dataset consists of multiple tables:
* customer_info → demographics
* income → annual income
* behavior → spending, purchases, discounts
* membership → membership duration
* churn → churn status

## Tools Used
* SQL (MySQL)
* Excel (for initial exploration)

##Key Insights
* High-value customers (~20–25%) contribute significantly to overall purchases
* ~35–40% of customers churn, with lower engagement levels
* Higher discount usage leads to ~15–25% increase in purchases
* Loyal customers (>5 years) show higher spending and purchase frequency
